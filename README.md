# Humidity Measurement System (Analog - LM324 Comparator Based)

This project is a cost-effective humidity measurement system using analog electronics. It uses two NTC thermistors (wet and dry method) along with an LM324 comparator to detect humidity variations.

## Features
- No microcontroller required
- Low-cost and energy-efficient design
- Simple and reliable analog circuit
- Easy to build and maintain

## Components Used
- LM324 Op-Amp (Comparator)
- 2 × NTC Thermistors (Wet & Dry)
- 3 × LEDs
- 3 × 220Ω Resistors
- 2 × 10kΩ Resistors
- 9V Battery
- Switch
- Wires, Soft Cloth, Rubber Holder

## Working Principle
One NTC thermistor is kept dry while the other is kept wet using a soft cloth. Due to evaporation, the wet thermistor cools down and its resistance changes.

This difference between the wet and dry thermistor is given as input to the LM324 comparator. Based on the difference, the output LEDs indicate humidity levels.

## Applications
- Environmental monitoring
- Agricultural applications
- Low-cost sensing systems

## Cost Estimation
The total cost of this project is approximately ₹130–₹150, making it an affordable solution for basic humidity sensing.

## Key Advantage
This system eliminates the need for a microcontroller, making it simple, cost-effective, and suitable for basic real-world applications.
