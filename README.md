# Humidity Measurement System (Analog - LM324 Comparator Based)

This project is a cost-effective humidity measurement system using analog electronics. It uses two NTC thermistors (wet and dry method) along with an LM324 comparator and potentiometers for calibration to detect humidity variations.

## Features
- No microcontroller required
- Low-cost and energy-efficient design
- Adjustable sensitivity using potentiometers
- Simple and reliable analog circuit

## Components Used
- LM324 Op-Amp (Comparator)
- 2 × NTC Thermistors (Wet & Dry)
- 3 × Potentiometers
- 3 × LEDs
- 3 × 220Ω Resistors
- 2 × 10kΩ Resistors
- 9V Battery
- Switch
- Wires, Soft Cloth, Rubber Holder

## Working Principle
One NTC thermistor is kept dry while the other is kept wet using a soft cloth. Due to evaporation, the wet thermistor cools down and its resistance changes.

The difference between the wet and dry thermistor is fed into the LM324 comparator. Potentiometers are used to adjust the reference levels and sensitivity.

Based on the comparison, LEDs indicate different humidity conditions.

## Applications
- Environmental monitoring
- Agricultural systems
- Low-cost sensing applications
